Template

Attributes

text[R]

Template text.

file[R]

File name of template, if given.

stencil[R]

Templating format to apply “whole-clothe”.

default[R]

Default format(s) for undecorated parts. If not otherwise set the default is ‘html’.

common[R]

Common format(s) to be applied to all parts. These are applied at the end.

header[R]

Header data, also known as front matter.
