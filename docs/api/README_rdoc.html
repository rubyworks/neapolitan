Neapolitan

  • home: rubyworks.github.com/neapolitan

  • code: github.com/rubyworks/neapolitan

DESCRIPTION

There are many markup and templating systems in the world. Why be limited to just one? Neapolitan gives you a whole box to pick from.

FEATURES/ISSUES

  • All the variety of a Whitman’s Sampler.

  • And all the ease of a Hershey’s K.I.S.S.

  • Website uses pretty colors ;)

SYNOPSIS

For now please see the Neapolitan website and API documentation.

HOW TO INSTALL

You know the routine ;) 

  $ sudo gem install neapolitan

If you’re old fashion and want to install to a site location, see Setup.rb (protuils.github.com/setup).
