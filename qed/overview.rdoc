= Overview of Using Chocolates

Require the library.

  require 'chocolates'

Load our example template[example.cocoa].

  template = "example.cocoa"

  factory = Chocolates::Factory.new(template)

Render with hash table.

  options = { :name=>"Tom" }

  factory.render(options)

Render in an Object scope.

  scope = Struct.new(:name).new("Becky")

  factory.render(scope)

Render via a binding.

  name = "Huck"

  factory.render(binding)
