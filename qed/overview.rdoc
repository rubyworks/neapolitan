= Overview of Using Chocolates

Require the library.

  require 'chocolates'

Load our example template[example.cocoa].

  tmpfile = "example.cocoa"

  options = {:name=>"Tom"}

  document = Chocolates::Document.new(tmpfile, options)

  document.to_s

Render in an Object scope.

  scope = Struct.new(:name).new("Becky")

  document.resource(scope)

  document.to_s

Render via a binding.

  name = "Huck"

  document.resource(binding)

  document.to_s
